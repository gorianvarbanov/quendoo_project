Overview

The Quendoo PMS API v1 provides a comprehensive interface for integrating with the Quendoo Property Management System. This RESTful API allows you to manage properties, availability, bookings, and more.

Base URLs:
DEBUG: https://quendoo-platform.itti.bg/api/pms/v1/
PRODUCTION: https://www.platform.quendoo.com/api/pms/v1/

Authentication

All API requests require authentication using an API key passed as a GET parameter.

Required Parameters

Parameter Type Required Description
api_key string Required Your API authentication key
api_lng string Optional Language code: en-GB, bg-BG, el-GR, de-DE, ro-RO, ru-RU

Example Request

GET https://www.platform.quendoo.com/api/pms/v1/Property/getPropertySettings?api_key=your-api-key&api_lng=bg-BG

Response Structure

All API responses return a JSON object with up to three possible keys: data, error, and meta.

{
  "data": {
    "rooms": [
      {
        "id": 1,
        "name": "Double Room"
      }
    ]
  }
}
{
  "error": {
    "code": 400,
    "message": "Bad Request",
    "details": "Not found property room_id"
  }
}
{
  "meta": {
    "message": "Success"
  }
}

Property Settings

Retrieve property data including rooms, rate plans, extra services, payment methods, meal types, beds, booking modules, and channel codes.

GET Property/getPropertySettings

Parameters

Parameter Type Required Description
names string Optional Comma-separated list to filter results (e.g., "rooms,rates")

Example Request

GET /Property/getPropertySettings?api_key=your-api-key&names=rooms,rates

Response Example

{
  "data": {
    "rooms": [
      {
        "id": 2666,
        "name": "Single Room"
      },
      {
        "id": 2667,
        "name": "Double Room"
      }
    ],
    "rates": [
      {
        "id": 3685,
        "name": "Standard Rate"
      }
    ]
  }
}

Rooms Details

Get detailed information about rooms in the property.

GET Property/getRoomsDetails

Parameters

Parameter Type Required Description
room_id integer Optional Specific room ID to get details for

Example Request

GET /Property/getRoomsDetails?api_key=your-api-key&room_id=2667

External Property Data

Submit property data from an external system to Quendoo for mapping purposes.

POST Property/postExternalPropertyData
Note: You must provide an id and name for the property, plus at least one structure: rooms_types, rates_types, services_types, meals_types, or beds_types.

Request Body Example

{
  "property": {
    "id": "p1",
    "name": "Hotel Enterprise",
    "mapping_data": {
      "rooms_types": [
        {
          "id": "r1",
          "name": "Double Room"
        }
      ],
      "rates_types": [
        {
          "id": "rp1",
          "name": "Early Bird"
        }
      ]
    }
  }
}

Success Response

{
  "meta": {
    "message": "Success"
  }
}

Get Availability

Retrieve available room quantities for a date range. Quendoo works at the room type level, not individual physical rooms.

GET Availability/getAvailability

Parameters

Parameter Type Required Description
date_from date Required Start date (YYYY-MM-DD)
date_to date Required End date (YYYY-MM-DD)
sysres string Required System identifier: "qdo" (Quendoo IDs) or "ext" (External IDs)
room_id integer Optional Specific Quendoo room ID
ext_room_id string Optional Specific external room ID

Example Request

GET /Availability/getAvailability?api_key=your-api-key&date_from=2026-01-15&date_to=2026-01-20&sysres=qdo

Response Example

{
  "data": {
    "2667": {
      "2026-01-15": 43,
      "2026-01-16": 41,
      "2026-01-17": 40
    }
  }
}

Update Availability

Modify available room quantities for specific dates.

POST Availability/updateAvailability

Request Body Example

{
  "values": [
    {
      "room_id": 2667,
      "date": "2026-01-15",
      "qty": 40
    },
    {
      "room_id": 2667,
      "date_from": "2026-01-20",
      "date_to": "2026-01-25",
      "qty": 35,
      "opened": true
    },
    {
      "ext_room_id": "ext-123",
      "date": "2026-01-18",
      "closed": true
    }
  ]
}
Parameter Options:
  • qty: Set available quantity
  • closed: Close room (true/false)
  • opened: Open room (true/false)

Get Bookings

Retrieve list of unconfirmed new or modified bookings with check-in dates greater than or equal to today.

GET Booking/getBookings

Parameters

Parameter Type Required Description
type string Optional "new" for new bookings only, "modified" for modified bookings only
Important: Each booking must be acknowledged using Booking/ackBooking with its revision_id, or it will continue to be sent.

Booking Status Values

  • status: REQUESTED, CREATED, APPROVED, CHECKIN, CHECKOUT, CANCELLED
  • payment_status: WAITING, FAILED, DEPOSIT_PAID, PARTIALLY_PAID, FULLY_PAID, REFUNDED

Example Request

GET /Booking/getBookings?api_key=your-api-key&type=new

Response Example (Abbreviated)

Acknowledge Booking

Confirm receipt of a booking to prevent it from being sent again.

POST Booking/ackBooking

Request Body Example

{
  "booking_id": 412687,
  "revision_id": "fa01cc5f58b3dddea6293a286b860fe4",
  "booking_items": [
    {
      "booking_item_id": 492319,
      "ext_reservation_id": "EXT-12345",
      "self_chekin_code": "ABCD1234",
      "room_number": "301"
    }
  ]
}
The booking_items array is optional but allows you to provide external reservation IDs, self check-in codes, and room numbers.

Room Assignment

Assign room numbers and self check-in codes to booked items.

POST Booking/postRoomAssignment

Request Body Example

{

      "room_number": "205"

}

Booking Offers

Retrieve booking offers based on search criteria including dates, guests, and booking module.

GET Property/getBookingOffers

Parameters

Parameter Type Required Description
bm_code string Required Active booking module code
date_from date Required Check-in date (YYYY-MM-DD)
nights integer Required Number of nights
guests array Required Guest configuration per room
currency string Optional BGN, EUR, USD, GBP, RUB
promo_code string Optional Promotional discount code

Guest Parameter Format

guests[0][adults]=2&guests[0][children_by_ages][0]=5&guests[0][children_by_ages][1]=8

Example Request

GET /Property/getBookingOffers?api_key=your-api-key&bm_code=neXu98qdmw&date_from=2026-01-15&nights=3&guests[0][adults]=2¤cy=BGN

Response Example

{
  "data": [
    {
      "per_room": false,
      "sell_type": "per_occupancy",
      "room_id": 2667,
      "room_name": "Double room",
      "rate_id": 8444,
      "rate_name": "Early bird",
      "price": 675,
      "currency": "BGN",
      "accommodation_name": "2 adult on regular bed"
    }
  ]
}
